# 🎬 Anime Engine API

Motor de geração de histórias e prompts de anime (cinematográfico, viral e consistente).

---

## 🔥 O que isso faz

- Gera histórias estruturadas em cenas
- Cria personagens consistentes (anime 2D)
- Gera prompts para imagem (Leonardo, PixAI)
- Gera prompts para vídeo
- Mantém continuidade entre cenas

---

## 🧠 IA usada

- OpenAI (GPT)
- Claude (planejado)

---

## ⚙️ Como funciona

1. Usuário envia uma ideia base  
2. API processa  
3. Retorna:
   - História
   - Personagens
   - Mapa de cenas
   - Prompts (imagem + vídeo)
